# Operation Boomithran - Google Sites Build Guide

## 1. Create the site

1. Go to [Google Sites](https://sites.google.com/new).
2. Select **Blank**.
3. Set the site name to **Operation Boomithran Timeline**.
4. In **Themes**, choose a clean theme. Set the header/background to deep blue and use pink for buttons/dividers where available.
5. Add a header title: **Operation Boomithran**.
6. Add subtitle: **Student Satellite Mission Timeline | TCE Space Club**.

## 2. Upload photos

Upload the image folders from this package to the same Google Drive folder first. Then use **Insert > Images > Drive** when building each section.

## 3. Add sections in this exact order

## 4. Finish and publish

1. Add a final dark-blue section: **The Journey Continues...**
2. Add text: **Next: Design - Build - Launch | TCE Space Club**.

4. Set the viewer access to the audience you want, then share the published URL.
